# 🗓️ Interactive Wall Calendar Component

A fully responsive, visually rich **Wall Calendar UI** built using **HTML, Tailwind CSS, and Vanilla JavaScript**.
This project transforms a static calendar design into a **functional, interactive, and user-friendly component** with modern UX features.

---

## 🚀 Live Demo

> *(https://kg0420.github.io/Frontend-Development-internship-task/)*

---

---

## 📌 Features

### 🎨 Wall Calendar Aesthetic

* Clean, modern UI inspired by a physical wall calendar
* Integrated **hero image + angled overlay design**
* Balanced layout between **visual content and functional calendar grid**

---

### 📅 Date Range Selection

* Click or **drag to select a date range**
* Visual states include:

  * Start date
  * End date
  * Highlighted range between them
* Smooth transitions and hover feedback

---

### 📝 Integrated Notes System

* Add notes for selected date ranges
* Notes are:

  * **Context-aware (linked to selected range)**
  * **Persisted using localStorage**
* Automatically loads saved notes for previously selected ranges

---

### 📱 Fully Responsive Design

* **Desktop:** Side-by-side layout (image + calendar panel)
* **Mobile:** Stacked layout for seamless touch interaction
* Optimized spacing, typography, and usability across screen sizes

---

## ✨ Creative Enhancements (Standout Features)

* 🖱 **Drag-to-select range interaction** (Google Calendar-like UX)
* 🌙 **Dark Mode Toggle** with persistent preference
* 🎨 **Dynamic Theme Switching** (image + color adaptation)
* 🔄 **Month Navigation with Flip Animation**
* 📍 **Today's Date Highlight**
* ⚡ Smooth transitions and micro-interactions

---

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS (via CDN)**
* **Vanilla JavaScript (ES6)**
* **LocalStorage API**

---

## 🧠 Design & Engineering Decisions

* Built using **Vanilla JS** to keep the solution lightweight and dependency-free
* Modular logic structure allows easy migration to frameworks like React
* Tailwind CSS used for:

  * Rapid UI development
  * Consistent design system
  * Responsive layout handling
* LocalStorage used for persistence to avoid unnecessary backend complexity
